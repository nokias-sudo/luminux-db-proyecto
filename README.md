# primera adminsitración de base de datos

Este documento es un backup de la base de datos de la biblioteca "luminux", que consta de cinco tablas cada una con sus respectivos atributos,
Las tablas de encuntran como: libros, prestamos, clientes, empleados y editoriales.

cada una cuenta con 25 datos ingresados, a exepción de la tabla empleados, que cuenta con 6 unicamente.

Se relacionan de la siguiente manera: libros----prestamos, clientes----prestamos, empleados----prestamos, editoriales----libros
